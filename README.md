# ZenVT

Десктоп приложение для OBS на Python - **ZenVT (Zen VTuber)**. Заменяет веб-камеру на анимированного чиби-персонажа, реагирующего на ваш голос.

## Возможности

- 🎙️ **Обнаружение речи** через микрофон (анализ громкости + pitch)
- 🎭 **Анимация спрайтов**: смена глаз (открыты/закрыты) и рта (открыт/закрыт)
- 🌊 **Левитация** аватара (настраиваемая)
- 📳 **Дрожь** при разговоре (интенсивность и частота)
- 👁️ **Моргание** в режиме ожидания
- 🎛️ **Большое количество настроек**: аудио, эффекты, спрайты, позиция/размер
- 📱 **Работа в трее** с быстрым доступом
- 🎨 **Тёмная тема** интерфейса

## Установка

### 1. Требования

- Python 3.10+
- Windows / Linux / macOS

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск

```bash
python main.py
```

## Настройка OBS

1. Запустите приложение - появится прозрачное окно с аватаром
2. В OBS добавьте источник **"Захват окна" (Window Capture)**
3. Выберите окно **"ZenVT Avatar"**
4. Включите **"Захват прозрачности"** в свойствах источника (или используйте хромакей в настройках ZenVT)
5. Настройте размер и позицию через окно настроек приложения

> **Совет**: Для лучшего результата используйте **"Game Capture"** вместо "Window Capture" - меньше артефактов.

## Загрузка спрайтов

1. Откройте настройки (иконка в трее → "Настройки")
2. Перейдите во вкладку **"Спрайты"**
3. Загрузите PNG-файлы с прозрачным фоном:

| Спрайт | Описание |
|--------|----------|
| `eyes_open_mouth_closed` | Персонаж смотрит, рот закрыт |
| `eyes_open_mouth_open` | Персонаж смотрит, рот открыт |
| `eyes_closed_mouth_closed` | Персонаж моргает, рот закрыт |
| `eyes_closed_mouth_open` | Персонаж моргает, рот открыт |

**Рекомендации к спрайтам:**
- Формат: PNG с прозрачным фоном
- Размер: минимум 512x512 пикселей
- Персонаж должен быть в центре

## Настройки

### Вкладка "Аватар"
- **Размер окна**: ширина и высота в пикселях
- **Позиция**: X/Y координаты окна на экране
- **FPS**: частота обновления анимации
- **Левитация**:
  - Вкл/Выкл
  - Скорость (циклов в секунду)
  - Амплитуда (пикселей)

### Вкладка "Аудио"
- **Микрофон**: выбор устройства ввода
- **Порог громкости**: чувствительность к речи (1-100)
- **Задержка тишины**: время тишины перед переходом в idle
- **Определение продолжительных звуков**: распознаёт "ааааа" как речь

### Вкладка "Эффекты"

#### Эффект "Говорит"
- **Дрожь**: вкл/выкл, интенсивность, частота
- **Синхронизация рта**: открытие рта пропорционально громкости

#### Эффект "Не говорит"
- **Плавный переход**: длительность анимации при завершении речи
- **Моргание**: интервал и длительность

### Вкладка "Спрайты"
- Выбор PNG-файлов для каждого состояния персонажа

## Структура проекта

## troubleshooting

### Окно аватара не видно в OBS
- Проверьте что окно не свёрнуто
- Включите "Захват прозрачности" в OBS
- Попробуйте другой метод захвата (Game Capture)

### Персонаж не реагирует на голос
- Проверьте выбор микрофона в настройках
- Уменьшите порог громкости
- Проверьте что микрофон работает в системе

### Спрайты не загружаются
- Убедитесь что файл PNG с прозрачностью
- Проверьте путь к файлу (без кириллицы и спецсимволов)
